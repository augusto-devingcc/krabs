# Code of conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

- Treat everyone respectfully. Disagreement is fine; personal attacks are not.
- Assume good faith. Most bad interactions are misunderstandings, not malice.
- Keep discussion technical and on-topic.
- No harassment, discrimination, or insults — in issues, PRs, commits, code, or anywhere else this project lives.

## Enforcement

If someone is violating this, email `support@krabs.dev`. We'll respond within 48 hours. Repeat offenders get banned from the repo.

## Scope

This applies to all project spaces: GitHub issues, PRs, discussions, code, commit messages, and anywhere someone is acting on behalf of the project.
